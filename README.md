 # Тестовое задание для Effective Mobile (DevOps)
Простое веб-приложение, состоящее из Python HTTP-сервера и прокси-сервера Nginx, развернутых в Docker-контейнерах.

## Как запустить
Запустите проект одной командой - docker-compose up -d --build

## Как проверить работоспособность
Выполнить запрос к локальному серверу - curl http://localhost

Ожидаемый ответ - `Hello from Effective Mobile!`

## Архитектура проекта
Nginx: Слушает порт 80 и перенаправляет трафик на сервис `backend`.
Backend: HTTP-сервер на Python, работающий внутри Docker на порту 8080. Порт закрыт для внешнего доступа и доступен только для Nginx.
Healthcheck: В Docker compose настроена проверка состояния бэкенда. Nginx не запустится, пока бэкенд не станет `healthy`.
Безопасность: Приложение в контейнере backend запускается от имени пользователя `http-server`.
